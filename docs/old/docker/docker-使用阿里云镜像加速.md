docker 的官方镜像在国外，国内环境访问速度慢，并且可能失败。 因为可以考虑使用 阿里云的 docker 镜像服务如何使用镜像服务，参考 阿里云的[《容器镜像服务 Docker 镜像的基本使用》](https://yq.aliyun.com/articles/70756) 即可。

## 一、操作步骤

1.  刚进来需要设置一个密码
2.  点击菜单栏的 『镜像加速器』
3.  然后根据操作文档，进行配置即可
